Diese Dokumentation beschreibt die Softwarearchitektur von *StadtratWatch* —
einer Plattform für Bürgerbeteiligung und Transparenz im Magdeburger Stadtrat.
Sie folgt dem https://arc42.org[arc42]-Template für Architekturdokumentation.

Jedes Kapitel liegt in einem eigenen Verzeichnis mit einer `index.adoc` und kann
sowohl eigenständig als auch über dieses Hauptdokument gelesen werden.
